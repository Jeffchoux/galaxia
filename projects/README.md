# Pièces du bâtiment Galaxia

Ce dossier contient les projets intégrés comme "pièces" dans Galaxia OS.
Chaque sous-dossier est un projet autonome avec son General Manager dédié.

Structure attendue (peuplée au fil des phases) :
- `learn-ai/`      - plateforme d'apprentissage IA (Milan, Phase 8)
- `worldseed/`     - AGI auto-gouvernée (Phase 8)
- `eterna-app/`    - plateforme de legs numériques (Phase 8)
- `boostmybiz/`    - SaaS à reconstruire via Phase 14 "création par phrase"

Les pièces sont peuplées en Phase 8 (migration des projets actifs)
et Phase 8.5 (discovery GitHub + provisioning interactif).
